## Codebase Patterns
- `evidence-demo/` is the TypeScript package; run `npm run typecheck`, `npm run test`, and `npm run build` from that directory
- Use `.js` extensions in TypeScript import paths (`moduleResolution: NodeNext`)
- Pure analyzers live in `src/analyzers/`; impure git/file I/O lives in `src/inputs/`; report assembly in `src/report/`
- CLI entry point is `src/cli.ts` â†’ `dist/cli.js` via `bin` in package.json
- History window constant `HISTORY_WINDOW_MONTHS = 6` lives in `gitHistorySource.ts`; use `historyWindowSince()` to compute the `since` date for queries
- `createGitHistorySource(repoPath).query({ authorEmail, path, since })` returns `{ authorCommitCount, totalCommitCount, lastTouchDate }` via local `git log`
- Tests live in `evidence-demo/test/`; compile with `tsconfig.test.json` to `dist-test/`; run via `node --test dist-test/test/**/*.test.js`
- `resolveChangedFiles` in `changedFiles.ts` uses local git only: `base...head` ranges, branch names (vs default branch), or PR numbers (via locally fetched `refs/pull/N/head`)
- Pure familiarity analyzer in `src/analyzers/familiarity.ts` injects `GitHistorySource`; use optional `asOf` date param for deterministic tests; `countAuthorCommitsToFile` is Slice 1, `analyzeFamiliarity` is Slice 2 (directory aggregation + recency)
- `touchedAreaForPath()` maps touched files to directory areas (`src/foo.ts` â†’ `src/`); `analyzeFamiliarity` deduplicates areas and queries history at directory level
- `shareOfAreaChurn()` and `characterizeFamiliarity()` implement Slice 3 normalization/labels; high requires last touch â‰¤60 days plus (â‰Ą3 commits or â‰Ą25% share); recency gates prevent stale history from yielding high
- `createImportGraph(repoPath)` in `importGraphSource.ts` walks `.ts`/`.tsx` files (skips `node_modules`, `dist`, etc.), parses imports via the TypeScript AST, resolves relative specifiers (extensionless + index files), and returns reverse-dependency map `ImportGraph` (target â†’ sorted importer paths)
- Pure blast-radius analyzer in `src/analyzers/blastRadius.ts` injects `ImportGraph`; `countDirectImportersForFile` is Slice 1 (single-file dependent count + paths); `analyzeBlastRadius` is Slice 2 (US-009)
